Grid Example

This is a container with a maxium width of 1200px with 40px padding on each side. The gutter gap is 20px.

        $column-count:          12;
        $container-width:       1200px; // Not including outside gutters.
        $width--gutter:         20px;
        $width--gutter-half:    $width--gutter / 2;
        $outside--gutter:       40px;
